# Recipe-website

# 🍽️ Meal Finder

**Meal Finder** is a web application that allows users to **search for meals**, view ingredients, and watch recipe videos. It fetches data from [TheMealDB API](https://www.themealdb.com/) and dynamically displays meal details.

## 🚀 Live Demo
🔗 [View Live Demo](https://recipe-website-6k81.onrender.com)

---

## 🎯 Features
✅ **Fetches meals dynamically** from TheMealDB API.  
✅ **Displays meal details** including name, image, and description.  
✅ **Lists ingredients** with measurements.  
✅ **Allows users to watch recipe videos** via YouTube.  
✅ **Mobile-friendly & responsive UI.**  

---

## 🛠️ Technologies Used
- **HTML**
- **CSS** (Flexbox & Responsive Design)
- **JavaScript (ES6+)**
- **Fetch API** for retrieving meal data from TheMealDB
